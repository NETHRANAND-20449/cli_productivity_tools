## **Where to Find These Tools:**

### **Primary Sources:**

1. **Official Arch Repositories**
   - `pacman -Ss <tool_name>`
   - Core, Extra, Community repos

2. **AUR (Arch User Repository)**
   - Browse: https://aur.archlinux.org
   - Use AUR helpers: `yay -Ss <tool>` or `paru -Ss <tool>`

3. **GitHub Awesome Lists**
   - https://github.com/alebcay/awesome-shell
   - https://github.com/k4m4/terminals-are-sexy
   - https://github.com/rothgar/awesome-tuis
   - https://github.com/toolleeo/cli-apps
   - https://github.com/agarrharr/awesome-cli-apps
   - https://github.com/unixorn/awesome-zsh-plugins
   - https://github.com/emilwallner/Reinforcement-Learning-Notebook

4. **Arch Wiki**
   - https://wiki.archlinux.org/title/List_of_applications
   - Category-specific pages (e.g., "File managers", "Music players")

5. **Reddit Communities**
   - r/commandline
   - r/archlinux
   - r/linux
   - r/unixporn (for configs)

6. **Terminal Trove**
   - https://terminaltrove.com
   - Curated terminal apps

7. **AlternativeTo**
   - https://alternativeto.net
   - Filter by "Linux" and "Command Line"

8. **Repology**
   - https://repology.org
   - Cross-distro package search

9. **Package Managers (Language-Specific)**
   - **Rust/Cargo**: https://crates.io
   - **Python/pip**: https://pypi.org
   - **Node/npm**: https://npmjs.com
   - **Ruby/gem**: https://rubygems.org
   - **Go**: `go install` from GitHub

10. **Snap/Flatpak/AppImage**
    - snapcraft.io
    - flathub.org
    - AppImageHub

### **Discovery Tools:**

- **`pkgstats`** - See what packages Arch users install most
- **`pacgraph`** - Visualize package dependencies
- **`pacman -Qe`** - List explicitly installed packages
- **`pacman -Qdt`** - Find orphaned packages

### **Trending & New Tools:**

- **GitHub Trending**: https://github.com/trending (filter by language: Shell, Rust, Go, Python)
- **Hacker News**: Search "Show HN: CLI tool"
- **Product Hunt**: Filter by "Developer Tools"
- **Changelog.com**: Podcast about open source tools

### **Specialized Directories:**

- **Rust CLI Working Group**: https://rust-cli.github.io
- **Modern Unix**: https://github.com/ibraheemdev/modern-unix
- **Command Line Heroes**: https://www.redhat.com/en/command-line-heroes

### **Learning Resources:**

- **tldr pages**: https://tldr.sh
- **cheat.sh**: `curl cheat.sh/<command>`
- **explainshell.com**: Explain shell commands
- **Man pages online**: https://man.archlinux.org
